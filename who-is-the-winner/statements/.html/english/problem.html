Who is the winner?
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

$$$Mazen$$$ and $$$Mahmoud$$$ are playing $$$Rock$$$ $$$Paper$$$ $$$Scissors$$$.

You will be given two characters, $$$A$$$ $$$Mazen's$$$ choice and $$$B$$$ $$$Mahmoud's$$$ choice.

Can you determine who will be the winner?

Game rules.

Input

Only one line contains $$$A$$$ and $$$B$$$ $$$\in$$$ $$$\{'R','P','S' \}$$$ where,

$$$'R'$$$ for $$$Rock$$$.

$$$'P'$$$ for $$$Paper$$$.

$$$'S'$$$ for $$$Scissors$$$.

Output

Output "Mazen" if he is the winner or "Mahmoud" if he is the winner or "Tie" otherwise, without double quotes.

Examples

Input
R S
Output
Mazen
Input
P S
Output
Mahmoud
Input
P P
Output
Tie
